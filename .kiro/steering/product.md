# Product Overview

E-commerce platform for a parrot supplies store. Customers can browse products across four categories (feed, cages, toys, vet supplies), add items to cart, and place orders with various delivery and payment options.

## Key Features
- Product catalog with filtering, sorting, and search
- Shopping cart with persistent state (Zustand)
- Multi-step checkout with delivery and payment options
- Order creation with Telegram webhook integration (planned)
- Responsive design with Tailwind CSS and shadcn/ui components

## Target Users
Pet owners shopping for parrot supplies online, primarily Russian-speaking market (UI text in Russian).
