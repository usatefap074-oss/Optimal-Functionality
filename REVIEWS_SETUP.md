# Компонент отзывов - Инструкция по использованию

## Что было добавлено

Новый компонент карусели отзывов с фотками попугаев от реальных клиентов и их отзывами о доставке, содержании и качестве.

## Структура

### База данных
- Новая таблица `reviews` в `shared/schema.ts` с полями:
  - `customerName` - имя клиента
  - `city` - город
  - `rating` - рейтинг (1-5 звёзд)
  - `text` - текст отзыва
  - `image` - URL фотографии попугая
  - `deliveryMethod` - способ доставки
  - `createdAt` - дата создания

### Компоненты
- `client/src/components/ReviewsCarousel.tsx` - карусель отзывов с навигацией
- `client/src/hooks/use-reviews.ts` - хук для получения отзывов с сервера

### API
- `GET /api/reviews` - получить все отзывы
- `POST /api/reviews` - создать новый отзыв (для админ-панели)

### Скрипты
- `script/seed-reviews.ts` - скрипт для добавления примеров отзывов

## Как использовать

### 1. Создать таблицу в БД
```bash
npm run db:push
```

### 2. Добавить примеры отзывов
```bash
npm run db:seed-reviews
```

### 3. Запустить приложение
```bash
npm run dev
```

Отзывы будут отображаться на главной странице в новой секции "Отзывы наших клиентов".

## Как добавить новый отзыв

Через API (например, из админ-панели):

```bash
curl -X POST http://localhost:5000/api/reviews \
  -H "Content-Type: application/json" \
  -d '{
    "customerName": "Иван Петров",
    "city": "Москва",
    "rating": 5,
    "text": "Отличный попугай! Очень доволен покупкой.",
    "image": "https://example.com/photo.jpg",
    "deliveryMethod": "Курьер"
  }'
```

## Особенности

- Карусель автоматически циклируется
- Адаптивный дизайн (мобильная и десктопная версии)
- Отзывы отсортированы по дате (новые первыми)
- Звёздный рейтинг визуально отображается
- Информация о способе доставки и городе клиента
